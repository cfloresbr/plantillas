Plantilla Recursos de Aplicativos ServiceIT

Descargas:






La plantilla cuenta con componentes de librería Boostrap v.5.3, jQuery v3.7 y fontawesome.

Esta plantilla se usa para:

- [SGA] Sistema Gestión de Asistencia

- [SRA] Sistema Registro de Asistencia

- [SRAI] Maestro de Gestión de Personas

- [MPU] Maestro de Población Usuaria

- [MR] Maestro de Recintos

- [OM] Ondamedia

- [RAFLEC] Fomento Lector

- [SGC] Sistema Gestión deConvenios

- [SGI] Sistema de Gestión Integral

- [SGPI] Sistema Gestión de Proyectos de Investigación

- [SIRAC] Sistema de Información para la Reportabilidad de las Actividades Culturales
